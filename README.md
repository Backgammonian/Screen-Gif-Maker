# Screen-Gif-Maker
WPF app for creating GIFs from screenshots.
## How to use:
* Start taking screenshots with given time interval
* Skip or move selected frames
* Set animation interval
* Select cropping area or don't select it at all
* Hit the button `"Create GIF"`
* Wait for the animated image to be created
* If you changed your mind you can cancel this process
* When process is done, you can open the file and enjoy the result :)
